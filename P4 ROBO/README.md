# Instructions
To run on Linux,
```
javac -cp "RobotSim.jar:JGameGrid.jar:" filename.java
java -cp "RobotSim.jar:JGameGrid.jar:" filename
```

To run on windows,
```
javac -cp "RobotSim.jar;JGameGrid.jar;" filename.java
java -cp "RobotSim.jar;JGameGrid.jar;" filename
```
